Each fighting style is treated as a Skill. Like all skills, it can be leveled up, to a maximum of 6

Attack an enemy and apply the Default Effect of the Fighting Style. Unless otherwise specified, the range for Striking a target is the Default Striking Range for this style.

Damage

Deal your Damage to an enemy.

Default

Apply the Default Effect of the Fighting Style.

Damage+

Deal your Damage to an enemy. and Apply Default

Techniques

Techniques are divided into 2 categories.

  • The Base Technique of the Fighting Style that the character learns as soon as he learns the fighting style.
  • Techniques learned as he progresses in the fighting style. When a character reaches a new level in a fighting style, he will have to chose 1 Technique between all techniques of his new level.

Base Technique

The same for almost all Fighting Styles.

Passive Abilities

These passive abilities apply only to Techniques of this Fighting Style. These are passive effect that are always active, unless otherwise specified.

Actions

These Techniques give access to new actions that can be used in Combat.

Global Passives

Upon reaching level 3 and level 6 in the Fighting Style. The character unlocks an equippable passive abilities.
These abilities can be equipped/unequipped each time the character has the time to rest for a while.
remark This is up to the GM really, if you want your players to have more or less freedom in swapping their Global Passives around it's all up to you. For me these represent a change in weapon used / enhancement of the character through various means.

Most of these Global Passives come with a Passive Slot Cost that are gained through character/equipment/story progression. For more information see Character Progression
